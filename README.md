A bioinformatics pipeline for identifying metallothionein (MT) genes, including homologues and paralogues, in Oryza coarctata using sequence similarity and genome-wide analysis.

📁 Repository Structure

├── data/              # Input FASTA / genome files

├── scripts/           # Python or shell scripts

├── results/           # Output files (BLAST hits, candidates)

├── notebooks/         # Colab/Jupyter notebooks (optional)

└── README.md
